НГТУ, ФБ, WEB-программирование, часть 2. Фильтры в Jinja2

Пример использования фильтров в Jinja2

Исходная фраза:

{{ phrase }}

Фильтр title:

{{ phrase|title }}

Фильтр striptags:

{{ phrase|striptags }}

Фильтр safe:

{{ phrase|safe }}

Фильтр replace:

{{ phrase|replace('сколько', 'много')|replace('нам', 'тут') }}

Фильтр upper:

{{ phrase|upper }}

Фильтр striptags + reverse:

{{ phrase|striptags|reverse }}
